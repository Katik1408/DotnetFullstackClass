CSS Selector Combination

1. Combine two Classes
.selector-1.selector-2{
property:value;
property:value;
}

Example:-
<h1 class="large-heading">Title</h1>

h1.large-heading{
font-size: 200%;
}


2. Combine ID and Classes

#id-selector.class-selector-1.class-selector-2{
property:value;
property:value;
}

<h2 id="big-blue" class="large blue">
Title
</h2>

#big-blue.large.blue{
color: blue;
font-size: 200;
}


3. Combile Ancestor and Child

.ancestor .child {
property:value;
property:value;
}
